
# Image Captioning English

This project creates captions for images in English.

## How to Run

1. Make sure you have Python 3.8 or newer installed. You can get it from [python.org](https://www.python.org/downloads/).

2. Open a terminal in the project folder.

3. Create a virtual environment (optional, but recommended):
   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   ```

4. Install the required packages:
   ```powershell
   pip install -r requirements.txt
   ```

5. Run the program:
   ```powershell
   python ImageCaptioning.py
   ```

6. Follow the instructions shown in the terminal. The script will ask for your input and show the captions.

## Tips

- Make sure your images are in a place the script can find.
- If you have problems, check the messages in the terminal.
